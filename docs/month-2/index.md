# Month 2: Deep Learning

With your ML foundations in place, this month goes deeper into neural network architectures. You'll learn how CNNs process images, how RNNs handle sequences, and how to train models that generalize well using modern optimization and regularization techniques.

## What You'll Cover

| Lesson | Topic |
|--------|-------|
| 1 | [Convolutional Neural Networks](cnns) — convolutions, pooling, and image classification |
| 2 | [RNNs & LSTMs](rnns-lstms) — sequential data, vanishing gradients, and gated architectures |
| 3 | [Loss Functions & Optimizers](loss-and-optimizers) — cross-entropy, Adam, learning rate schedules |
| 4 | [Regularization](regularization) — dropout, weight decay, batch norm, data augmentation |
| 5 | [Transfer Learning](transfer-learning) — using pretrained models, fine-tuning strategies |
| 6 | [**Project: Sentiment Analysis**](month2-project) — an LSTM-based classifier on IMDB reviews |

## Prerequisites

All of Month 1, especially neural network fundamentals and your first model.

## Capstone Project

You'll build an LSTM-based sentiment classifier that predicts whether movie reviews are positive or negative, practicing data preprocessing, model design, regularization, and evaluation.

## Estimated Time

**~60–80 hours** total (15–20 hours/week over 4 weeks).
