# Month 3: NLP & Transformers

This is where everything changes. You'll learn how machines process language — from basic text preprocessing through word embeddings, sequence-to-sequence models, and the attention mechanism — culminating in a ground-up understanding of the Transformer architecture that powers every modern LLM.

## What You'll Cover

| Lesson | Topic |
|--------|-------|
| 1 | [Text Preprocessing](text-preprocessing) — tokenization, cleaning, vocabulary building |
| 2 | [Word Embeddings](word-embeddings) — Word2Vec, GloVe, and learned representations |
| 3 | [Sequence-to-Sequence Models](seq2seq) — encoder-decoder architectures for translation |
| 4 | [The Attention Mechanism](attention-mechanism) — how models learn to focus |
| 5 | [The Transformer Architecture](transformer-architecture) — self-attention, multi-head attention, positional encoding |
| 6 | [Transformer from Scratch](transformer-from-scratch) — building a complete Transformer in PyTorch |
| 7 | [**Project: Text Classifier with Transformers**](month3-project) — Transformer encoder vs. LSTM baseline |

## Prerequisites

Months 1–2, especially RNNs/LSTMs from Month 2 (the Transformer is motivated by their limitations).

## Capstone Project

You'll build a text classification model using a Transformer encoder, train it on the IMDB dataset, and compare its performance against an LSTM baseline.

## Estimated Time

**~60–80 hours** total (15–20 hours/week over 4 weeks).
