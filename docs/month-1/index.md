# Month 1: AI & ML Foundations

This month builds your foundation. You'll understand what AI is, learn the math it speaks, set up your Python environment, grasp the core ideas of machine learning, and build your first neural network from scratch.

## What You'll Cover

| Lesson | Topic |
|--------|-------|
| 1 | [What Is Artificial Intelligence?](what-is-ai) — history, types, and where the field is today |
| 2 | [Math Foundations](math-foundations) — linear algebra, calculus, and probability for ML |
| 3 | [Python for AI](python-for-ai) — NumPy, data manipulation, and plotting |
| 4 | [Intro to Machine Learning](intro-to-ml) — supervised vs. unsupervised, bias-variance, evaluation |
| 5 | [Neural Networks Introduction](neural-networks-intro) — perceptrons, backpropagation, activation functions |
| 6 | [Your First Model](your-first-model) — building and training a neural network end to end |
| 7 | [**Project: MNIST Digit Classifier**](month1-project) — a complete image classification pipeline |

## Prerequisites

- Basic programming experience (any language; Python preferred)
- High-school math (algebra, basic statistics)
- Python 3.10+ installed on your machine

## Capstone Project

You'll build a neural network that classifies handwritten digits from the MNIST dataset with >97% accuracy. This covers the full pipeline: data loading, preprocessing, model architecture, training, evaluation, and experimentation.

## Estimated Time

**~60–80 hours** total (15–20 hours/week over 4 weeks).
